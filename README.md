# Projeto-de-extensao

O sistema foi desenvolvido em Python com interface gráfica (Tkinter) e banco de dados MySQL para ajudar na organização de uma atividade de extensão sobre inclusão digital.

Ele permite cadastrar participantes, registrar presença e coletar avaliações sobre a oficina, como notas e comentários. Todas as informações ficam salvas no banco de dados para facilitar o controle.

O sistema também gera relatórios simples com a quantidade de participantes, presença e média das avaliações, ajudando a entender os resultados da atividade realizada na comunidade.

É um programa simples desenvolvido no VSCOde para um trabalho de faculdade, então ele não roda diretamente pelo github.

***IMPORTANTE***

Para rodar o código é necessário executar ele no seu terminal ou compilador de código, eu recomendo o VSCode pois o programa foi desenvolvido diretamente nele.
